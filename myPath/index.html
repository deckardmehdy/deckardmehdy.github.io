Welcome to myPath!

This is a quick tutorial that demonstrates how myPath works. If you want to skip this, then press the 'X' in the upper right-hand corner.
What are pathfinding algorithms?

Pathfinding algorithms are a class of algorithms that find the shortest path from one node to another on a given graph.


What is myPath?

myPath is a visualizing tool that shows how different pathfinding algorithms work.

The program's objective is to go from the start node to the end node using a pathfinding algorithm. The program can only move horizontally or vertically, with each movement having a cost of one.

Once the program is complete, the shortest path will be illuminated in yellow.


Step 1: Choose An Algorithm

Click the 'Algorithms' tab and select one of the algorithms from the dropdown menu.


Step 2: Draw Walls

Click or hold your mouse down over cells to create walls.


Step 3: Press Start

Press the Start button in the upper right-hand corner to start your pathfinding algorithm!

Optional: Mazes

Click the 'Mazes' tab and select one of the algorithms from the dropdown menu and a randomized maze will be created for you!
